Connect four

Challenge a friend to disc-dropping fun with the classic game of Connect 4! Drop your red or yellow discs in the grid and be the first to get 4 in a row to win. If your opponent is getting too close to 4 in a row, block them with your own disc! Whoever wins can press the replay button to release all the discs and start the fun all over again!.

Cannon

Cannon is a shooting game in which you control the Core Cannon, which moves horizontally across the screen firing at aliens. The aim is to shoot all of the opponents in the rows.Arrow keys are used to move the cannon on the horizontal axis and space bar to shoot those opponents.

Memory Match

Memory match is a fun card game in which cards are dealt and the player reacts quickly in the least amount of time possible to spot pairs of cards of the same rank.

RPS

Rock paper scissors is a hand game usually played between two players, in which each player simultaneously forms one of three shapes with an outstretched hand. These shapes are "rock", "paper", and "scissors".
